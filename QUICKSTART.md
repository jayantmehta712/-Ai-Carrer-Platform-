# Career AI Platform - Project Initialization Guide

## Quick Start

### 1. Backend Setup
```bash
cd backend
npm run dev
```
Backend will run on: http://localhost:5000

### 2. Frontend Setup
```bash
cd frontend
npm run dev
```
Frontend will run on: http://localhost:5173

## Project Overview

This is a full-stack AI-powered career guidance platform with:

- **Frontend**: Modern React application with TypeScript
- **Backend**: Node.js/Express REST API with TypeScript

## Next Steps

1. **Environment Configuration**
   - Copy `.env.example` to `.env` in backend folder
   - Add your API keys and database configuration

2. **Database Setup**
   - Configure MongoDB connection or use the provided placeholder

3. **Feature Development**
   - Start implementing career recommendation features
   - Add authentication and user management
   - Integrate with AI models (OpenAI, etc.)

4. **Testing**
   - Add unit tests for services
   - Add integration tests for API endpoints
   - Add E2E tests for user flows

## Architecture

- **Components**: Modular, reusable React components
- **Services**: Centralized API communication
- **Types**: Strongly typed with TypeScript
- **Routing**: RESTful API design
- **Styling**: Tailwind CSS for responsive design
